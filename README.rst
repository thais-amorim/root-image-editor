Root Image Editor
========

First project of the Digital Image Processing course.

Install dependencies
--------------------

$ pip install -r requirements.txt

Running
-------

$ python3 ./scripts/main.py

Features
--------

Implemented filters:

- Negative
- Logarithmic
- Gamma correction
