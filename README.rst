Root Image Editor
========

First project of the Digital Image Processing course.

Install dependencies
--------------------

$ pip install -r requirements.txt

Running
-------

$ python3 ./scripts/main.py

Testing
-------

$ pytest

Features
--------

Implemented filters:

- Negative
- Logarithmic
- Gamma correction
- Histogram Equalization
- Median

Other:

- Draw histogram of any image
